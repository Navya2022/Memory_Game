# MemoryGame

It is an interactive flashcard-based memory matching game built using HTML, CSS, and JavaScript. Users can create custom flashcards using text or images and test their recall through a matching gameplay interface. The project also includes a preset mode for quick play without manual input.

---

## Features

- **Custom Flashcards**
  - Supports both text and image-based cards
  - User-generated inputs are stored using `localStorage`

- **Preset ("Lazy") Mode**
  - Pre-loaded cards for quick, no-setup gameplay

- **Interactive Flip & Match Interface**
  - Front–back card animation
  - Matched cards remain flipped; unmatched cards reset

- **Minimal, Lightweight, Browser-Based**
  - No backend required
  - Works offline

---

## Technology Stack

| Component | Technology |
